Final Visualization // Report Data description The data are collected by The Common Core of Data (CCD). The primary purpose of the CCD is to provide basic information on public elementary and secondary schools, local education agencies (LEAs), and state education agencies (SEAs) for each state. The table used in this project contains 18 variables (include, state, school name, student race, student gender, student count, etc.) and 11,149,338 records of students’ information across the U.S. at school level.
